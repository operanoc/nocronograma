Cero Infraestructura

Un solo archivo HTML. Sin servidor, sin base de datos, sin mantenimiento. Se sirve desde GitHub Pages.

  • Destacar la simplicidad de la arquitectura: un archivo HTML que hace todo
  • Explicar que GitHub Contents API actúa como base de datos sin costo
  • Mencionar que el audit trail permite saber exactamente quién modificó qué
  • Resaltar que no se necesita ningún tipo de infraestructura o mantenimiento
